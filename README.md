# Application Form

This is a multi step form with React for animal adoption. 

### Technologies

I used:

* React.js
* React create app
* ES6
* Material UI

### Setup

To run this project:

```sh
$ npx create-react-app .
$ npm i material-ui 
$ npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
